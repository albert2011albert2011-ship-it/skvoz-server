# Мессенджер "Сквозь" 🌐

Децентрализованный мессенджер, работающий **с интернетом и без него** через Bluetooth, Wi-Fi Direct и Mesh-сеть.

## Особенности ✨

### 🔥 Ключевые возможности
- **Работа без интернета** - общайтесь через Bluetooth и Wi-Fi Direct
- **Mesh-сеть** - сообщения могут проходить через другие устройства (А → Б → В → Г)
- **Автоматический выбор соединения** - приложение само выбирает лучший способ доставки
- **Полная конфиденциальность** - нет центрального сервера, все данные только на устройствах
- **Передача файлов** - отправляйте фото, документы и другие файлы
- **Красивый дизайн** - современный Material Design 3 интерфейс

### 📡 Типы подключений
1. **Bluetooth** - прямое соединение с ближайшими устройствами (до 100м)
2. **Wi-Fi Direct** - высокоскоростное соединение (до 200м)
3. **Mesh Relay** - многоскачковая передача через других пользователей
4. **Интернет** (опционально) - для расширения радиуса действия

### 🔒 Конфиденциальность
- ❌ Нет регистрации по email/телефону (необязательно)
- ❌ Нет центрального сервера
- ❌ Нет облачного хранения сообщений
- ✅ Все данные хранятся только на вашем устройстве
- ✅ Сообщения передаются напрямую между устройствами

- Android 6.0+ / iOS 12.0+

## Как это работает 🛠️

### Mesh-маршрутизация
Приложение использует алгоритм **flooding с отслеживанием путей**:

1. **Обнаружение соседей** - каждое устройство периодически рассылает свой профиль
2. **Таблица маршрутов** - устройства запоминают, через кого можно достичь других
3. **Пересылка пакетов** - сообщения передаются всем соседям (кроме тех, кто уже в пути)
4. **TTL (Time To Live)** - ограничение на количество прыжков для предотвращения бесконечной пересылки

### Пример работы
```
Пользователь А хочет отправить сообщение Пользователю Г

Прямой связи нет, но есть цепочка: А ↔ Б ↔ В ↔ Г

1. А отправляет сообщение (TTL=10, путь=[А])
2. Б получает, добавляет себя в путь, уменьшает TTL (путь=[А,Б], TTL=9)
3. В получает, добавляет себя (путь=[А,Б,В], TTL=8)
4. Г получает - сообщение доставлено!
```

## Использование 📱

### Первый запуск
1. При первом запуске введите имя и никнейм
2. Email и телефон указывать необязательно
3. Примите политику конфиденциальности
4. Нажмите "Войти"

### Общение
1. На главном экране отображаются все пользователи в сети
2. Нажмите на пользователя для открытия чата
3. Вводите сообщения и отправляйте
4. Статусы сообщений показывают путь доставки

### Работа офлайн
- Включите Bluetooth и/или Wi-Fi
- Приложение автоматически найдет ближайшие устройства
- Сообщения будут доставлены через цепочку устройств

## Технологии 💻
